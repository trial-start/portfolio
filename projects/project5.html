Tours App

Empowering users with seamless exploration, Tours harnesses a meticulously crafted RESTful API built on Express.js, MongoDB, and Mongoose. Featuring intuitive CRUD functionalities for tours and reviews, the project prioritizes data integrity and security, ensuring a frictionless experience

Enriching user journeys with polished aesthetics, Tours integrates the Pug template engine to deliver an unparalleled interface. Leveraging Express.js middleware, the project guarantees robustness by enforcing meticulous request validation, data sanitization, and error handling, culminating in a seamless and secure exploration platform.

Technologies used: Node.js, Express.js, MongoDB and pug templating engine
