CineSuggest

CineSuggest is a movie recommendation application built using python, stramlit, ML, seaborn and matplotlib. It utilizes Item based collaborative filtering technique to give recommendations.

The UI contains a search bar where users can enter the name of a movie to receive recommendations. It also features a sidebar providing information about the app and dataset statistics, and a slider to select the number of recommendations. The recommendations are displayed with visual representations, including bar plots for distances, count of ratings, and average ratings for each recommended movie, enhancing user interaction and experience.

Technologies used:Streamlit, Python, pandas, matplotlib, seaborn
