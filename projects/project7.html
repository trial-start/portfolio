getMovies

getMovies redefines movie management with rating and watch list features. Using HTML, CSS, JavaScript, and React, it seamlessly integrates the OMDB API for comprehensive movie details. Users effortlessly access, rate, and organize movies into personalized watch lists, aided by robust local storage management.

This intuitive platform empowers users to curate their movie collections with ease. Leveraging efficient local storage and OMDB API integration, "getMovies" provides streamlined access to detailed movie data, enhancing the personalized movie-watching experience.

Technologies used: React.js, HTML, CSS
